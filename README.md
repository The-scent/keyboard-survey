# ⌨️ 键盘选型助手

一个帮助客户快速找到适合自己键盘的交互式选型工具。通过回答 14 个简单问题，AI 推荐引擎自动匹配最合适的键盘型号。

## 功能

- **智能推荐引擎** — 根据预算、使用场景、配列偏好、轴体手感等 14 个维度匹配键盘
- **98 款键盘数据库** — 覆盖 150 元以下到 3000 元以上各价位段
- **一键购买** — 每款推荐键盘附带京东搜索链接
- **实时修改** — 在结果页直接修改任意选项，推荐即时刷新
- **结果分享** — 通过链接或图片分享推荐结果给朋友
- **多配列支持** — 全尺寸 / 98键紧凑 / TKL / 75% / 65% / 60% / Alice人体工学

## 使用

访问 https://the-scent.github.io/keyboard-survey/ 即可使用。

答题完成后可以：
- 复制结果链接发给朋友
- 生成结果图片分享到微信
- 点击顶部答案标签修改选项

## 技术栈

纯前端单页应用（HTML + CSS + JavaScript），无后端依赖，部署于 GitHub Pages。

- 评分匹配算法，含预算约束惩罚机制
- URL hash 编码实现结果分享（Base64 + encodeURIComponent）
- Canvas 生成结果分享图片
- localStorage 存储答题历史
